# AI 塔罗牌占卜应用

## 项目概述
这是一个基于Python和PyQt5开发的AI塔罗牌占卜应用，结合了传统塔罗牌的神秘智慧与现代AI的分析能力，为用户提供个性化的塔罗牌解读体验。

## 功能特点
- 支持抽取1-10张塔罗牌进行占卜
- 提供多种牌阵选择（单牌、三牌、五牌、七牌、十牌）
- 包含78张完整的塔罗牌（大阿卡纳和小阿卡纳）
- 每张牌都有详细的含义解释
- 基于OpenAI API的智能解读功能
- 历史记录保存与查看功能
- 复制牌面信息到剪贴板
- 美观友好的用户界面

## 项目结构
```
aiapp/
├── main.py             # 程序入口
├── tarot_deck.py       # 塔罗牌核心逻辑
├── gui.py              # 用户界面
├── ai_analysis.py      # AI分析功能
├── requirements.txt    # 项目依赖
├── config.json         # 配置文件
└── README.md           # 项目说明
```

## 安装指南
1. 克隆或下载项目到本地
2. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```
3. 在`config.json`文件中填入您的OpenAI API密钥
4. 运行`main.py`启动应用：
   ```
   python main.py
   ```

## 使用方法
1. 在输入框中输入您的问题
2. 选择要抽取的牌数量
3. 点击"抽取塔罗牌"按钮
4. 等待AI分析完成后查看结果
5. 可以点击"历史记录"查看过往占卜结果
6. 可以点击"复制牌面"将牌面信息复制到剪贴板

## 注意事项
- 本应用需要有效的OpenAI API密钥才能使用AI解读功能
- 请确保您的网络连接正常
- 塔罗牌解读仅供参考，请勿过分依赖

## 依赖项
- PyQt5: 5.15.9
- openai: 0.27.8
- python-dotenv: 1.0.0
